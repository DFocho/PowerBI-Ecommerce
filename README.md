# E-Commerce Sales and Customer Behavior Analysis (Power BI)

## Overview

This project analyzes ecomerce transaction data to uncover revenue trends, customer behavior and profuct performance.

## Datasets

- ecom_customers.csv: customer-level data
- ecom_order_items.csv: items ordered, quantity, price
- ecom_orders.csv: order-level data
- ecom_products.csv: product-level data. Product name, category

## Objectives

- Analyze revenue trends and growth
- Identify top-perfroming products
- Understanding customer purchasing  behavior

## Key KPIs

- Total revenue
- Average order value (AOV)
- Orders per Customer
- Category-wise revenue

## Dashboard Features

- Revenue trend over time
- category performance breakdown
- Customer purchase patterns
- Order distribution analysis

## Key Insights

- A small number of categories drive the majority of revenue
- Higher order values are associated with specific product types
- Repeat customers contribute significantly to total revenue

## Business Recommendations

- Focus marketing efforts on high-performing categories
- Increase customer retention through loyalty programs
- Bundle products to increase average order value

## Tools Used
- DAX
- PowerBI

## Takeaway

This project demonstrates how transaction data can be leveraged to improve revenue and customer experience in e-commerce.
